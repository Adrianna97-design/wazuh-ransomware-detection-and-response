# wazuh-ransomware-detection-and-response
Zaawansowany scenariusz detekcji ransomware i automatycznej reakcji w Wazuh dla środowiska hybrydowego (Windows + Linux). Repozytorium zawiera scenariusz detekcji, niestandardową regułę Wazuh, konfigurację Active Response (izolacja hosta, blokada procesu) oraz integrację alertów z Teams/Slack i systemami ticketowymi.
